# 快速开始

## 🚀 立即使用

### 第一步：运行脚本
双击 `txt_cleaner.py` 文件（或在终端运行 `python3 txt_cleaner.py`）

### 第二步：输入路径
```
请输入TXT文件所在目录路径：[输入或粘贴你的文件夹路径]
请输入处理结果输出目录路径：[输入你想保存结果的路径]
```

### 第三步：确认处理
输入 `y` 并按回车，开始批量处理

## 📁 文件说明

- **txt_cleaner.py** - 主程序脚本
- **使用说明.md** - 详细使用文档
- **处理示例/** - 示例处理结果（使用上传的文件作为测试）

## 💡 使用技巧

**Windows用户：**
按住 Shift，右键点击文件夹 → "复制为路径" → 粘贴

**Mac用户：**
直接拖拽文件夹到终端窗口获取路径

**建议：**
首次使用建议先用测试文件试运行，确认效果满意后再处理重要文件
