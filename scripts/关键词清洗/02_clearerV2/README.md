# 贴吧文本批量清理工具

## 快速开始

### Windows用户
1. 双击 `启动清理工具.bat`
2. 按提示输入输入文件夹路径和输出文件夹路径
3. 确认后等待处理完成

### macOS/Linux用户
1. 打开终端
2. 运行: `python3 tieba_text_cleaner.py`
3. 按提示输入路径

## 文件说明

- **tieba_text_cleaner.py** - 主程序脚本
- **启动清理工具.bat** - Windows一键启动器
- **使用说明.md** - 详细使用说明
- **README.md** - 本文件

## 功能

批量清理贴吧帖子TXT文件中的:
- 签到信息
- 广告内容
- 系统提示
- 界面元素
- 其他格式文本

将所有格式文本替换为 `|` 字符,保留真实讨论内容。

## 支持的关键词

脚本已预置674个贴吧帖子常见的格式文本模式,包括:
- 日一二三四五六
- 签到排名相关(含数字匹配)
- 广告文本
- 楼层管理信息
- IP属地
- 社交分享链接
- 等等...

详见 `使用说明.md`
