# 快速开始

## 运行步骤

1. **运行脚本**
   ```bash
   python txt_processor.py
   ```

2. **输入信息**
   - 第一步：输入TXT文件所在目录的路径
   - 第二步：输入处理结果保存的目录路径

3. **完成**
   - 脚本会自动处理所有TXT文件
   - 处理结果保存在指定的输出目录

## 示例

## 处理规则

### 规则1：合并特殊相邻行
如果发现以下情况的相邻两行：
- 一行只有竖线（|、||、|||等）
- 另一行是空行或只有空格

→ 将这两行合并为一个空行

### 规则2：压缩连续空行
如果发现3行或更多的连续空行：

→ 压缩为2个空行

## 提示

- ✓ 支持子目录（会自动查找所有子目录中的TXT文件）
- ✓ 保持目录结构（输出文件会保持与输入相同的目录结构）
- ✓ 不修改原文件（所有处理结果保存在新目录）
- ✓ 支持UTF-8编码的TXT文件
