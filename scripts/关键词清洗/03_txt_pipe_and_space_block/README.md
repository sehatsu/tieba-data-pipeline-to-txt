# TXT文件批量处理工具

这是一个专门用于批量处理TXT文件的Python脚本，可以自动清理和规范化文本文件格式。

## 📦 文件列表

- **txt_processor.py** - 主程序脚本
- **使用说明.md** - 详细使用说明
- **快速开始.md** - 快速入门指南
- **测试示例/** - 测试文件和示例

## ✨ 功能特点

1. **自动查找** - 递归查找指定目录下所有TXT文件
2. **智能处理** - 识别并处理特殊格式问题
3. **批量操作** - 一次性处理多个文件
4. **保持结构** - 输出文件保持原有目录结构
5. **安全可靠** - 不修改原文件，结果保存到新目录

## 🎯 处理规则

### 规则1：合并特殊相邻行
```
原文件：              处理后：
这是文本             这是文本
|||                  
                     这是文本
这是文本
```

### 规则2：压缩连续空行
```
原文件：              处理后：
这是文本             这是文本
                     
                     
                     这是文本
                     
                     
这是文本
```

## 🚀 快速开始

1. 确保已安装Python 3.x
2. 运行脚本：
   ```bash
   python txt_processor.py
   ```
3. 按提示输入输入和输出目录路径
4. 等待处理完成

## 📝 常见问题

**Q: 支持哪些文件格式？**
A: 仅支持 .txt 文本文件。

**Q: 会修改原文件吗？**
A: 不会，所有处理结果保存在输出目录，原文件保持不变。

**Q: 支持中文吗？**
A: 完全支持中文，包括全角空格的识别。
